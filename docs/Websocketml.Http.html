Module Websocketml.Http

module Http: sig .. end

Implement a subset of HTTP/1.1 RFC2616. Its only purpose is to do a websocket handshake, it cannot be used as regular HTTP server. One could use any other mean to retrieve an open socket and client address to use as inputs of Websocketml.create.

exception HTTPError of string
type t 
val create : Unix.sockaddr -> t

Create a TCP socket and bind it to given address

val listen : t -> unit

Set up the socket for receiving connection requests (with a pending list of size 1)

val accept : t -> Unix.file_descr * Unix.sockaddr

Block until a connection is received and return the socket and address of the connecting client

val listen_and_accept : t -> Unix.file_descr * Unix.sockaddr

Call listen then accept

val do_ws_handshake : Unix.file_descr -> unit

Read a WebSocket opening request on the given socket and answer with an appropriate message to complete the handshake defined by RFC6455.

Raise HTTPError if the received request is not a valid HTTP request or if it is not a valid WebSocket opening request.

val close : t -> unit

Close the server's TCP socket
